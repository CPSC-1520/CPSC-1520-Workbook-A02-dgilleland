CMIS1014 System Foundations
  • Requisites: This course is available to students in the Digital Media and IT program.
  • Open Studies: Not Available
  • Credits: 3.00

Students will learn how to work within the computer environment in use at NAIT, and access the software they need to complete their courses. This course includes learning how to set up common configurations of network architectures, network addressing schemes and protocols, including how these items are connected to their computer, and network-based activities such as browsing and file downloading. Students will learn how to employ backup methods and technologies to protect themselves and employers against data loss. In addition, students will learn about common operating system features such as binary numbering, the BIOS, environment variables, the registry and encoding methods.
