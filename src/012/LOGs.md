- [x] Define the term "truthiness" as it relates to JavaScript
  - "truthiness" is a term used to describe the "truth-like" value of a variable. In JavaScript, the following values are considered "falsey":
    - `false`
    - `0`
    - `""` (empty string)
    - `null`
    - `undefined`
    - `NaN`
